# Zepto Sales Analysis using SQL

## 📌 Project Overview
This project analyzes Zepto's product dataset using SQL to extract meaningful business insights.

## 📂 Dataset
- Zepto Product Dataset
- Contains product names, MRP, discount percentage, stock availability, category, etc.

## 🛠️ SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- CASE WHEN
- Subqueries

## 📈 Business Questions Solved
- Top 10 most expensive products
- Category-wise revenue estimation
- Out-of-stock product analysis
- Average discount by category
- Duplicate product detection
- Top discounted products
- Product count by category
